# ThreeMF Preview Privacy & Support

Static bilingual privacy and support pages for ThreeMF Preview, a macOS Finder and Quick Look 3MF preview tool.

## Pages

- Home: <https://chenhuanjie125.github.io/threemf-preview-support/>
- Privacy policy: <https://chenhuanjie125.github.io/threemf-preview-support/privacy.html>
- Support: <https://chenhuanjie125.github.io/threemf-preview-support/support.html>

## Publishing

GitHub Pages is configured to deploy the `main` branch from the repository root (`/`). The site is intentionally static: it uses HTML, CSS, and a small local language-switching script, with no analytics, login, database, or model upload service.
